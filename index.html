ML-Based Intrusion Detection System

A cutting-edge solution leveraging machine learning to enhance cybersecurity through real-time threat detection.

Problem

Modern cybersecurity demands real-time and intelligent intrusion detection systems (IDS) to combat evolving threats like malware, DDoS attacks, and MITM exploits.

Dataset

The system was trained and tested on diverse datasets including:

  • backdoor_malware
  • benigntraffic
  • ddos-syn_flood15
  • ddos-syn_flood1
  • MITM-arpspoofing1
  • reon-portscan

These datasets represent a range of malicious and benign network traffic scenarios.

Methods

The intrusion detection system employs a variety of machine learning algorithms, including:

  • Logistic Regression
  • Random Forest
  • Decision Tree
  • Linear SVC
  • K-Nearest Neighbors
  • Naive Bayes
  • AdaBoost
  • Gradient Boosting Classifier
  • Extra Trees Classifier
  • Ridge Classifier
  • Linear Discriminant Analysis
  • Neural Network

These models were implemented using scikit-learn, with Logistic Regression tuned for up to 500 iterations.
