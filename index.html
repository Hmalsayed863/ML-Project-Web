ML-Based Intrusion Detection System

A cutting-edge solution leveraging machine learning to enhance cybersecurity through real-time threat detection.

Problem

Modern cybersecurity demands real-time and intelligent intrusion detection systems (IDS) to combat evolving threats like malware, DDoS attacks, and MITM exploits.

Dataset

The system was trained and tested on diverse datasets including:

  • backdoor_malware
  • benigntraffic
  • ddos-syn_flood15
  • ddos-syn_flood1
  • MITM-arpspoofing1
  • reon-portscan

These datasets represent a range of malicious and benign network traffic scenarios.

Methods

The intrusion detection system employs a variety of machine learning algorithms, including:

  • RandomForestClassifier
  • AdaBoostClassifier
  • GradientBoostingClassifier
  • DecisionTreeClassifier
  • KNeighborsClassifier
  • SVC (Support Vector Classifier)
  • GaussianNB (Naive Bayes)
  • LogisticRegression

These models were implemented using scikit-learn, with LogisticRegression tuned for up to 500 iterations.

Results

Model performance was evaluated using key metrics such as:

  • Accuracy
  • Precision (weighted average)
  • Recall (weighted average)
  • F1-Score (weighted average)
  • Confusion Matrix

Results for each metric were formatted to four decimal places for precision. Detailed performance metrics will be updated soon.

Conclusion

The ML-based intrusion detection system demonstrates promising potential in identifying and mitigating network threats in real-time. By leveraging a diverse set of algorithms, the system achieves robust performance across various attack scenarios. Future work will focus on improving model accuracy and reducing false positives to enhance reliability in production environments.
